# 05 宿舍生活規定模板

## 使用情境

工廠或仲介需要通知外籍員工宿舍規定，避免生活習慣不同造成爭議。

## 中文公告

宿舍規定提醒：

- 晚上請降低音量
- 不要在房間抽菸
- 垃圾請依規定丟棄
- 公共區域使用後請清潔
- 外人進入宿舍需先告知
- 有設備故障請立刻回報

宿舍是大家一起住的地方，請互相尊重。

如果有問題，請先告訴主管或宿舍管理人。

## LINE 短版

宿舍請大家一起維持：

1. 晚上降低音量
2. 不在房間抽菸
3. 垃圾依規定丟
4. 公共區域用完要清潔
5. 設備壞掉要回報

請互相尊重。

## 萬語通使用提示

宿舍規定適合固定置頂在 LINE 群組，並保留多語版本。
