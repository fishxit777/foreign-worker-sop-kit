# 廠語通 FactoryTong SOP 包

Factory LINE SOP System for Foreign Worker Communication.

這是一個給工廠老闆、現場主管與人資使用的 Factory LINE SOP 模板包，用來處理外籍員工每天重複發生的現場溝通。

定位不是單純「翻譯工具」，而是把每天重複交代的事情整理成可複製、可翻譯、可留紀錄的 LINE SOP 管理流程。

## 適合誰

- 有越南、印尼、泰國、菲律賓等外籍員工的工廠
- 每天需要交代加班、集合、請假、排班、工安、SOP 的主管
- 需要降低語言誤會、減少重複說明的人資或老闆
- 想先整理管理流程，再導入萬語通的公司

## 產品差異

萬語通是工具，負責在 LINE 裡翻譯、OCR、語音、摘要與留紀錄。

廠語通 FactoryTong SOP 包是內容與導入包，負責幫主管先準備好「每天要講什麼」。

簡單說：

- FactoryTong SOP 包：準備公告內容、規則、交接模板
- 萬語通：把內容放進 LINE 群組裡翻譯、處理、追蹤

## 模板內容

- `templates/01-overtime-notice.md`：加班通知
- `templates/02-leave-policy.md`：請假規則
- `templates/03-safety-reminder.md`：工安提醒
- `templates/04-machine-sop.md`：機台 SOP 交辦
- `templates/05-dormitory-rules.md`：宿舍生活規定
- `templates/06-line-group-management.md`：LINE 群組管理規則

## 成交流程

1. 先用 Threads / IG 找到工廠老闆、主管、人資。
2. 不直接硬推 SaaS，先問「你們外籍員工公告、加班、請假都怎麼交代？」
3. 提供廠語通 FactoryTong SOP 包當低門檻入口。
4. 客戶用模板後，導入萬語通到 LINE 群組。
5. 用月費 SaaS 維持長期收入。
